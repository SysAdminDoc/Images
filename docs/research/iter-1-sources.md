# Windows Image Viewer Landscape Research - Sources (Iteration 1)

## (1) Direct OSS Competitors

- **ImageGlass 10 Official Website (imageglass.org)**
    *   Takeaway: Major rewrite to .NET and Avalonia UI for cross-platform support, improved performance, and large image handling.
- **nomacs Official Website (nomacs.org)**
    *   Takeaway: Cross-platform image viewer with robust RAW/PSD support, unique synchronization features, and basic image manipulation.
- **qView Official Website (interversehq.com/qview/)**
    *   Takeaway: Minimalist, fast, and lightweight cross-platform image viewer with GIF controls, basic manipulation, and URL opening.
- **JPEGView GitHub (github.com/sylikc/jpegview)**
    *   Takeaway: Fast, minimalist Windows image viewer and editor with real-time processing and extensive format/RAW support.
- **FastStone Image Viewer Official Website (faststone.org)**
    *   Takeaway: Feature-rich image browser, converter, and editor for Windows with extensive manipulation, batch processing, and slideshow capabilities, free for personal use.
- **Honeyview Official Website (bandisoft.com/honeyview/)**
    *   Takeaway: Fast, lightweight image viewer with broad format support including RAW and animated, archive viewing, EXIF/GPS display, and basic batch processing, though no longer actively updated (succeeded by BandiView).
- **Pictus Official Website (poppeman.se/pictus/)**
    *   Takeaway: Fast, lightweight, open-source Windows image viewer with minimalist UI, advanced preloading, wide format support, and Explorer integration for thumbnails.
- **Oculante GitHub Repository (github.com/woelper/oculante)**
    *   Takeaway: Free, open-source, hardware-accelerated image viewer/editor written in Rust, supporting a wide range of formats including professional ones, with non-destructive editing, image analysis, and network listen mode.
- **Lyn Official Website (lynapp.com)**
    *   Takeaway: High-performance image browser and viewer exclusively for macOS, featuring non-destructive editing, wide format support, library integration, geotagging, and metadata management.
- **XnView MP Official Website (xnview.com/en/xnviewmp/)**
    *   Takeaway: Powerful, cross-platform image viewer, browser, and converter supporting over 500 formats, offering advanced viewing, editing, batch processing, and creation tools, free for private use.

## (2) Commercial

- **Microsoft Windows - Photos App (microsoft.com/en-us/windows/photo-movie-editor)**
    *   Takeaway: Official Windows image viewer with AI-powered editing features (Generative Erase, Background Blur), multi-source gallery integration (OneDrive, iCloud), and organization tools like Memories and search.
- **Adobe Lightroom Classic Official Product Page (adobe.com/products/photoshop-lightroom-classic.html)**
    *   Takeaway: Professional-grade desktop-optimized photo editor and organizer (DAM) with advanced AI masking, content-aware removal, lens blur, and robust cataloging features.
- **IrfanView Official Website (irfanview.com)**
    *   Takeaway: Fast, compact, and long-standing Windows image viewer with extensive format support (images, video, audio), powerful batch processing, basic editing, and freeware for non-commercial use.

## (3) Adjacent

- **Squoosh Official Website (squoosh.app)**
    *   Takeaway: Google Chrome Labs' open-source, browser-based image optimization tool known for its real-time side-by-side comparison UX, advanced codec support (AVIF, WebP, MozJPEG), and local, offline processing.
- **darktable Official Website (darktable.org/features/)**
    *   Takeaway: Open-source photography workflow application and RAW developer for non-destructive editing, featuring high-precision 4x32-bit pixel buffers, GPU acceleration (OpenCL), professional color management, and robust organization tools.
- **digiKam Official Website (digikam.org/about/overview/)**
    *   Takeaway: Open-source professional photo management system (DAM) with extensive organizing features, advanced metadata support (AI-driven tagging, facial recognition), a powerful image editor, and comprehensive RAW support.
- **Upscayl Official Website (upscayl.org)**
    *   Takeaway: Free, open-source AI image upscaler using Real-ESRGAN models, offering up to 16x enlargement, local GPU processing, batch processing, multiple AI models, and custom model support.
- **chaiNNer Official Website (chainner.app)**
    *   Takeaway: Open-source, cross-platform visual node editor for building complex AI-powered image processing pipelines, supporting multiple AI backends, GPU acceleration, batch processing, and conditional logic.
